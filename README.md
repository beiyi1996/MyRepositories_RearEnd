# 後端 Windows Form 作品  
## 1. 票價運輸  
> + 使用技術 : C# － ADO.NET Entity Framework  
> + 目的 : 使用者可以選擇需要的車票類型，並將計算後的票價顯示在對應欄位上。

## 2. 生命靈數  
> + 使用技術 : C# － LINQ , DateTime型別  
> + 目的 : 依據使用者輸入的生日，計算出該使用者的生命靈數及對應的個性介紹。
